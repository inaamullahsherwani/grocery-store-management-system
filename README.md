# Grocery Store Management System

A web-based management system built with Flask and SQL to streamline grocery store operations across multiple branches.

## Tech Stack
- **Backend:** Python, Flask
- **Database:** SQL
- **Frontend:** HTML, CSS

## Features
- Manages 500+ grocery items across multiple store branches
- Inventory tracking with real-time stock updates
- Automated billing system
- KPI-based sales analytics dashboard
- Role-based access for admins and store managers

## Results
- Reduced manual stock-tracking time by 50%
- Helped stores increase monthly revenue by 15-20%
- Achieved 90% adoption rate among partnered stores
